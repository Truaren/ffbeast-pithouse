# FFBeast Pit House

Enhanced version of the original **FFBeast WheelHub**

This fork redesigns the interface and workflow to provide a **desktop experience similar to Moza Pit House**.  
The application is compiled as a **standalone .exe** for quick and convenient wheel configuration.

## Features

- **Moza-style interface** with grouped settings
- **Profile system** for different games (quick switching)
- **Steering angle control** and wheel centering
- **Motor tuning:** Encoder CPR, Pole Pairs, P Gain, I Gain
- **Force feedback settings:** motion range, damping, effect strength
- **Pedal binding and testing**
- **Calibration & safety settings**
- **Quick Save & Reboot**
- **DFU mode switch**
- **System tray support and autostart**
